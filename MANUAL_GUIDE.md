# AyurTwin User Manual - Documentation Guide

## Overview

The **AyurTwin User Manual** is a comprehensive, professional guide designed for both patients and family members who use the AyurTwin application. This 16-page PDF manual covers all aspects of the platform with clear instructions, examples, and troubleshooting guidance.
